How To Backup Databases Using mysqldump Tool

2018-05-28
Backup, DataBases, MySQL, SQL

source: http://www.mysqltutorial.org/how-to-backup-database-using-mysqldump.aspx

MySQL GUI tools, often provide features for backup MySQL dbs with ease

  • phpMyAdmin
  • SQLyog

Problem:
the backup process could be very slow if your db is big
because: the backup file needs to be transferred across the network to your client PC

Features of mysqldump:

  • best solution if your db is big
  • a very useful tool for backing up or dumping MySQL dbs locally on the server very fast
  • located in the root/bin folder of MySQL installation folder.
  • can be used to dump dbs for backup or transfer db to another db server
  • The dump file contains a set of SQL statements to create db objects.
  • can be used to generate CSV, delimited or XML files

How to backup a MySQL db (Export)
